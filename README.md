# react-fluxxor-sample

## Inatall

```
$ git clone https://github.com/hkusu/react-fluxxor-sample.git
$ cd react-fluxxor-sample
$ npm install
```

## Use

```
$ npm run build
```

or

```
$ $(npm bin)/gulp [build|watch|serve]
```
